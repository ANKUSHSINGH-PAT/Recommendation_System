# AI-Powered Recommendation System with MLOps & Kubernetes

## Overview

This project is an end-to-end AI-powered Recommendation System built using Machine Learning and deployed with modern MLOps practices. The application demonstrates model development, containerization, CI/CD, monitoring, and Kubernetes deployment.

The objective of this project is to showcase practical implementation of AI/ML engineering, cloud-native deployment, and production-ready MLOps workflows.

---

## Key Features

- AI-based Recommendation Engine
- Machine Learning model training and inference
- Python-based backend application
- Docker containerization
- Kubernetes (EKS) deployment
- CI/CD using GitHub Actions
- Prometheus monitoring
- Flask Web Application
- Modular project architecture
- Production-ready deployment workflow

### Clone Repository

```bash
git clone https://github.com/<your-username>/<repository-name>.git

```

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

## Docker

Build Image

```bash
docker build -t recommendation-system .
```

Run Container

```bash
docker run -p 5000:5000 recommendation-system
```
